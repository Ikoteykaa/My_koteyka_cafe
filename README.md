# ☕ My Koteykaa Cafe

A modern desktop application built with **C# and WPF** for creating, managing, saving, and loading cafe bills.

## 📖 Description

**My Koteykaa Cafe** is a WPF desktop application that allows users to create and manage customer bills in a simple and intuitive interface. The application supports adding and removing menu items, calculating tips and taxes, displaying the total bill, and saving or loading bills from text files. It focuses on usability, reliability, and data validation.

## ✨ Main Features

- ➕ **Add & Remove Items:** Easily add new menu items or remove selected ones.
- 💸 **Tip Calculation:** Supports both percentage-based and fixed amount tips.
- 🧾 **Bill Summary:** Automatically calculates Net Total, GST (5%), Tip, and Total.
- 🧹 **Clear Bill:** Quickly remove all items from the current bill.
- 💾 **Save Bill:** Save the current bill to a text file.
- 📂 **Load Bill:** Open a previously saved bill from a text file.
- 🛡️ **Input Validation:** Checks descriptions, prices, and file names before processing.
- ⚙️ **Error Handling:** Displays informative messages and prevents application crashes.
